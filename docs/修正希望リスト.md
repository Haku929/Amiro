# 修正希望リスト（優先度順）

## 1. フロントとAPIのつなぎ込み【高】
- チャット・分析・プロフィール・スロット・マッチングなど、まだモックや仮データのままの箇所を実APIに接続する

## 2. マッチングスロット：削除ボタン削除【高】
- マッチングスロットの削除ボタンを削除する

## 3. 共鳴マッチング：再検索設定パネルは不要【高】
- 再検索設定パネルは不要なので削除する（画面上部の分人の切り替えのみで十分）

## 4. チャット画面：フォーカス維持【高】
- メッセージを1通送るたびに入力欄のフォーカスが外れる → 送信後もフォーカスを維持する
- AIの返答が返ってくるまで送信できないようにする

## 5. チャット画面：「分析して終了」まわりが窮屈【高】
- 「終了して分析」ボタンあたりが窮屈に見える → ターン表示（○/10）を左側（シチュエーション横など）に移動し、ボタン側の余白を増やす

## 6. 凡例とプロットの統一（結果・レポート ＋ 共鳴詳細）【中】
- **結果・レポート画面（分人レポート）**: 凡例は白黒なのにプロットはカラーで、自分と相手の分人の区別が分かりづらい → 凡例とプロットの色・スタイルを統一する（凡例もカラー、白抜きなど検討）
- **共鳴詳細画面**: 同様に凡例は白黒でプロットはカラー → 凡例とプロットの色・スタイルを統一する（チャートコンポーネントの共通修正で両方対応可能）

## 7. 共鳴マッチング画面：Searching・レイアウト【中】
- **Searching 表示**: 分人を切り替えたときの「Searching」が上部にしかかからない → 画面全体、またはリスト部分（下側）にだけオーバーレイする
- **レイアウト**: 縦に並べると狭い → 左に「あなた」、右に「マッチング相手」を縦に並べ、2つのベクトル（レーダーチャート）も縦に並べるレイアウトを検討する

## 8. 共鳴詳細画面：共鳴スコアの色【中】
- スコアの値に応じて色が変わるようにする（例: 低→黄、中→緑、高→青など）

## 9. マッチングスロット：表示サイズ【中】
- 各ページのスロットが小さくて見づらい → カードやフォントなどを大きくして視認性を上げる

## 10. ユーザー情報の設定（名前・アイコン・自己紹介）【中〜低】
- 新規登録時とプロフィール画面の両方で、**名前（表示名）** と **アイコン（アバター）** を設定できるようにする
- **自己紹介** も設定・表示できるようにする（DB・API・UIの対応が必要）

## 11. プロフィールとホームの統合（検討）【低】
- 現状どちらも情報が少なくさびしい → プロフィール画面とホーム画面を統合する案を検討する（1画面にまとめるか、タブやセクションで分けるかは要検討）

## 12. AIのチャットプロンプトの改善
- 人間臭く、攻めたテーマにする
- ユーザー依存ではなくAIが話題を提供する
- 返答を短く（1〜3文 → 1文程度を検討）
- AI側の性格（鏡のBig5）に合わせたトーンにする
